
### 使用步骤


- 步骤1
  
![Step1](S11.png)

###### 该装置为翻盖式设计，打开盖子即可进行接下来的操作。


- 步骤2

![Step2](S2.png)

###### 按照图中的方式链接电源线与电池。

  
- 步骤3

![Step3](S3.png)

###### 将电源线另一头的2mm直径插头插入唯一的端口。

- 步骤4

![Step4](S4.png)

将一张任意的SIM卡按照图中的方向插入。随后合上翻盖，即可使用。

- 步骤5

![Step5](S5.png)

此按钮是装置上唯一的按钮，当报警发生时，按下以停止报警，程序也随之重新开始。
